# @formular/atomos Showcase

This is a Next.js application that serves as the primary showcase and interactive demonstration for the `@formular/atomos` library. It contains examples of all the components and their integration with `formular.dev` validation.

## Getting Started

First, install dependencies from the monorepo root:

```bash
npm install
# or
pnpm install
```

Then, run the development server from the monorepo root or within this package:

```bash
# From root
pnpm deploy:showcase

# From this package
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Purpose

The showcase is used to:
- Test components during development
- Demonstrate the capabilities of the library to users
- Provide an interactive reference for developers integrating the library
